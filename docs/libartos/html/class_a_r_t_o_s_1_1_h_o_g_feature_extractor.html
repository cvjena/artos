Detailed Description

Computes histograms of oriented gradients (HOG) as image features by delegating the main computations to the FFLD library.

The HOG features used for that implementation are described in "Object Detection with Discriminatively Trained Part Based Models" by Felzenszwalb, Girshick, McAllester and Ramanan, PAMI10.

To use another feature extractor by default, re-define FeatureExtractor in FeatureExtractor.h.
