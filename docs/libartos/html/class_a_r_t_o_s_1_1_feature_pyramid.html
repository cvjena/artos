Detailed Description

A pyramid of features extracted from multiple scales of an image using the default FeatureExtractor.

The scale of the pyramid level of index i is given by the following formula: 2^(1 - i / interval), so that the first scale is at double the resolution of the original image).

This class has been derived from FFLD's HOGPyramid.
